# Hana Beauty Medspa Website

Professional medical aesthetics spa website featuring advanced animations, mobile-responsive design, and comprehensive service pages.

## 🎨 Features
- Single Page Application (SPA) with custom router
- 16+ individual service landing pages
- Appointment scheduling interface
- Mobile-responsive design
- GSAP animations with ScrollTrigger
- Tailwind CSS styling
- SEO optimized

## 📦 Tech Stack
- Vanilla JavaScript + HTML5
- Tailwind CSS (CDN)
- GSAP Animation Library
- Playfair Display & Lato Typography

## 🚀 Deployment
This is a static website that can be deployed to Netlify, Vercel, GitHub Pages, or any static hosting service.
